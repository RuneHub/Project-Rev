# Project Rev.
### Current status: On-going, V0.75.
>[!NOTE]
>This is only a codebase without any necessary files, Downloading might give errors due to lack of necessary assets.


This is the codebase of my personal project, Project Rev, which is a third-person action game demo, where you fight a boss. 
This project was made with the intention of learning more about game development and explore the industry more thoroughly.

## Features:
- Fully Original 3D characters & Environment.
  * Characters designed and modeled using the "Arc System Works" method, which I learned from scratch during this project.
- Third Person Character Action Combat.
  * A take on a dual-pistol-wielding character in an action-driven game.
- Custom AI Behaviour.
  * A custom-built AI behaviour tree that makes decisions based on condition and situations.
- Handcrafted cinematics.
  * My First attempt at creating cinematics using Unity's Timeline.

## Tech Stack
- Game engine: Unity.
- Programming language: C#.
- 3D Modelling & Animations: Blender.
- Textures: Paint.net, Krita.
- Audio: Audacity.
